-თეორია-
1)რამდენი ცვლადი გვაქვს ჯავასკრიპტში.(პასუხი თეორიულად გაეცი)

    ჯავასკრიპტში გვაქვს 3 ცვლადი: let, const, var.
    
    let - არის block scope ცვლადი, რომლის მნიშვნელობის შეცვლა შესაძლებელია.

    const - არის არის block scope ცვლადი, რომლის მნიშვნელობის შეცვლა არ შეიძლება, რადგან
    არის კონსტანტა ანუ მუდმივა და მას ვერ "გადავაწერთ" მნიშვნელობას.


2)რამდენიტიპი გვაქვს ჯავასკრიპტში.(ჩამოთვალე)(პასუხი თეორიულად გაეცი)

    ჯავასკრიპტში გვაქვს ორი ტიპი, პრიმიტიული და არაპირიმიტიული.

    პრიმიტიულ ტიპებში შედის: string, number, boolean, null, undefined, bigint, symbol.
    არაპრიმიტიული ტიპი: objcet. (arrays, functions, date და ა.შ. შედის აქ).

3)რა მოვალეობა აქვს სკოუპის მეგობარ left hand side search - ს და right side search - ს.(პასუხი თეორიულად გაეცი)

    LHS(left hand side) -  როცა ცვლადს მნიშვნელობას ვანიჭებთ. მაგ: x = 5, აქ ხდება LHS
    ძიება,, რომ x ცვლადი იპოვოს ან შექმნას.
    RHS(right side search) - როცა ცვლადის მნიშვნელობას ვკითულობთ. მაგ (onsole.log(x); აქ ხდება RHS 
    ძიება, რათა ცვლადის მნიშვნელობა წაიკითხოს.

4) რომელი მეთოდი აქცევს სტრინგს მასივად.(პასუხი თეორიულად გაეცი)

    split() - მეთოდი.

5) სტრინგი პრიმიტიული ტიპია თუ არა ? .(პასუხი თეორიულად გაეცი)

    სტრინგი პრიმიტიული ტიპია.

6)რა განსხვავებაა == და === - შორის

    "==" ადარებს მნიშვნელობას, მაგალითად x = 5, და y = "5", ამ შედარების დროს ტოლია, ხოლო
    "===" შედარების დროს, მნიშვნელობასთან ერთად ტიპსაც ადარებს, ანუ x და y არ იქნება ერთმანეთს ტოლი,
    რადგან x-არის number, ხოლო y - string.
